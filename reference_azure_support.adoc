= 支持的 Azure Blob 访问层和区域
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Cloud Tiering 支持 _hot_ 访问层和大多数地区。



== 支持的 Azure Blob 访问层

在将数据分层设置为 Azure 时， Cloud Tiering 会自动对非活动数据使用 _hot_ 访问层。



== 支持的 Azure 区域

Cloud Tiering 支持以下 Azure 区域。



=== 非洲

* 南非北部




=== 亚太地区

* 澳大利亚东南部
* 东亚

* 日本西部

* 韩国
* 东南亚
